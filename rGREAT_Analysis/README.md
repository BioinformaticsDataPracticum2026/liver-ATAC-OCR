# rGREAT: GO Biological Process Enrichment Analysis of Open Chromatin Regions in Human and Mouse Liver

## Aim
The aim of this analysis is to identify candidate biological processes regulated by open chromatin regions (OCRs) in human and mouse liver tissue using GO Biological Process (GO:BP) enrichment analysis via rGREAT. rGREAT (Regulatory Genomic Regions Enrichment of Annotations Tool) associates genomic regions with nearby genes and then tests whether those genes are enriched for specific biological processes compared to a background expectation. By comparing enrichment results across species-shared and species-specific OCRs, we can determine which regulatory programs are evolutionarily conserved and which are unique to each species. This helps us understand how open chromatin landscape differences between human and mouse may translate into functional divergence at the level of biological pathways.

## Input Files
The input data for this analysis consists of open chromatin regions (OCRs) identified from human and mouse liver ATAC-seq data. The original peak files represent all accessible regulatory regions in each species' liver tissue. These OCRs were further classified into shared (orthologous regions open in both species) and species-specific (open in one species but closed in the orthologous region of the other species) groups using HALPER for cross-species liftover and BEDTools intersect for peak overlap classification. The original peak files are located in the `Mapping/outputs/` directory, while the shared and species-specific BED files are located in the `PE_classification/output/unique/` directory.

**Part a — Each species' OCRs (all open chromatin regions):**
- **Human all**: `Mapping/outputs/human_liver.narrowPeak`
- **Mouse all**: `Mapping/outputs/mouse_liver.narrowPeak`

**Part b — Shared OCRs (open in both species):**
- **Shared**: `PE_classification/output/unique/shared_open.bed`

**Part c — Species-specific OCRs (open in one species, closed in the other):**
- **Human specific**: `PE_classification/output/unique/human_open_mouse_closed.bed`
- **Mouse specific**: `PE_classification/output/unique/mouse_open_human_closed.bed`


### Method
rGREAT was used to perform GO:BP enrichment analysis on five sets of OCRs derived from ATAC-seq data. For each set of OCRs, the genomic coordinates were submitted to GREAT, which assigns each region to nearby genes based on regulatory domain associations and then performs a hypergeometric test to determine which GO:BP terms are significantly overrepresented among those genes compared to a genomic background.

The five OCR sets analyzed were as follows. First, **human all** represents all OCRs from the original human liver ATAC-seq peak file, providing a comprehensive view of biological processes regulated by open chromatin in human liver. Second, **mouse all** represents all OCRs from the original mouse liver ATAC-seq peak file, serving the same purpose for mouse. Third, **shared** contains HALPER-lifted human peaks that overlap native mouse liver peaks, which were generated using bedtools intersect. These represent orthologous OCRs that are open in both species and allow us to identify biological processes under conserved regulatory control. Fourth, **human specific** contains human-open/mouse-closed OCRs, representing regulatory regions that are accessible in human liver but closed in the orthologous mouse regions. Fifth, **mouse specific** contains mouse-open/human-closed OCRs, representing regulatory regions accessible only in mouse liver.

For each condition, the enrichment results were filtered to retain only statistically significant GO:BP terms based on adjusted p-value. The top 15 most significant terms per condition were visualized as horizontal bar plots, where bar length represents fold enrichment (how much more enriched the process is compared to background) and bar color represents -log10(adjusted p-value), with darker red indicating greater statistical significance. A comparison heatmap was also generated by taking the union of the top 10 most significant terms from each condition and displaying their fold enrichment values across all five conditions, allowing for direct visual comparison of which processes are enriched in which OCR sets.

### Scripts
- `scripts/rgreat_analysis.R` — Runs rGREAT enrichment analysis on each of the five OCR sets and outputs both full and significant GO:BP enrichment results as CSVs in the outputs directory.
- `scripts/rgreat_plots.R` — Reads the significant GO:BP CSVs and generates bar plots showing the top 15 enriched terms per condition, as well as a comparison heatmap across all conditions. Bar plots use fold enrichment on the x-axis and -log10(adjusted p-value) as the fill color.
- `scripts/run_rgreat.sh` — Shell script to execute the rGREAT analysis R script on Bridges-2.
- `scripts/run_plots.sh` — Shell script to execute the plotting R script on Bridges-2.
